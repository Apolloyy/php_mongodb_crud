# CRUD RapidCure Pharmacy

Sistema de Gerenciamento do estoque de produtos de uma farmácia apenas com o intuito de aprimorar minhas habilidades.

## Stacks utilizadas

**Front-end:** HTML, CSS e JS

**Back-end:** PHP

**Banco-de-Dados:** MongoDB

**Frameworks, libs:** Nenhuma (por enquanto)

**Icons:** https://iconsvg.xyz/
